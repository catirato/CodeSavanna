# CODESAVANNA

Esta aplicação foi desenvolvida em Java no âmbito de um trabalho prático de Programação Estruturada e simula a gestão de um zoo interativo chamado Code Savanna. O sistema funciona em consola e utiliza ficheiros CSV para armazenar informação sobre animais, clientes e interações, permitindo organizar e analisar os dados de forma simples e funcional.

A app inclui dois perfis de utilização: Administrador e Cliente. Na área de administração é possível consultar ficheiros, analisar estatísticas gerais, calcular receitas por tipo de interação, identificar o animal mais popular, descobrir as espécies com mais apadrinhamentos, listar padrinhos, encontrar o espetáculo mais rentável e consultar dados por habitat ou por nível de risco de extinção. Já na área de cliente, o utilizador pode explorar o catálogo de animais por habitat, consultar atividades de cada animal, simular apadrinhamentos e ainda jogar um mini-jogo de adivinhação de espécies.

Este projeto foi pensado para aplicar conceitos de leitura de ficheiros, matrizes, menus interativos, condições, ciclos e organização modular do código, combinando lógica de programação com um tema criativo e apelativo.
